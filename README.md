# Starbattle

### Features

- Daily games
- Free play boards
    - Records per board
    - Percent you beat
    - 8x8 1, 10x10 1, 10x10 2, 12x12 2, 14x14 3
- 1v1
    - ELO rating system
    - Paired with person online closet to your rating
    - Can challenge someone to a game at a specific time
- Leaderboards
    - By rating
    - By fastest times
- Light and dark modes
- User accounts
    - Profile pictures, bio
    - All time stats
- Game replays
